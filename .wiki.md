# Project Summary
The **BalloonPricer** project is a mobile-first application designed for balloon artists to streamline the pricing of balloon designs and related services. By allowing users to input parameters like balloon sizes and labor hours, the app generates detailed price breakdowns, enhancing pricing strategies. It also features design management capabilities, making it an essential tool for professionals in the balloon artistry business. Recent updates have focused on improving usability, particularly for mobile users, enhancing price input handling, and fixing navigation issues.

# Project Module Description
- **Calculator**: Main interface for calculating balloon design prices, featuring dropdowns for sizes and accessories, and a custom currency input.
- **Settings**: Configuration options for pricing, labor rates, balloon sizes, and logo uploads, optimized for mobile usability.
- **History**: Archive of previously saved designs with a searchable feature for easy management.
- **Authentication**: User sign-in and session management for personalized settings, including login and sign-up functionalities.
- **Header**: Displays the user's uploaded logo with mobile-friendly adjustments and a back navigation link.
- **Bottom Navigation**: Provides easy access to different app sections, optimized for mobile use, with improved z-index to prevent overlay issues.
- **AuthError**: Custom error page for handling authentication errors, styled to match the BalloonPricer branding.

# Technology Stack
- **Backend**: Python, FastAPI, SQLAlchemy
- **Frontend**: React, TypeScript, Tailwind CSS
- **Database**: PostgreSQL
- **Authentication**: JSON Web Tokens (JWT)
- **File Storage**: Object storage for logo uploads
- **PWA Support**: Web App Manifest for mobile-friendly features

# Usage
1. **Install Dependencies**:
   - Navigate to the backend directory and run:
     ```
     pip install -r requirements.txt
     ```
   - Navigate to the frontend directory and run:
     ```
     pnpm install
     ```

2. **Build the Frontend**:
   ```
   pnpm run build
   ```

3. **Run the Backend**:
   ```
   python app/backend/main.py
   ```

4. **Access the Application**: 
   Open your browser and navigate to the development server URL.
